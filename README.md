## 开发环境搭建

### 1. 克隆项目

`git clone https://github.com/takosenpai2687/topico-web.git`

### 2. 安装依赖

- `cd topico-web`
- `npm ci`

### 3. 启动项目

- （可选）在vscode中打开：`code .`

- `npm run dev`

## 代码提交规范

永远从 `master` 上新建分支，在新分支上开发（vscode左下角，GitHub Desktop都有显示）。

新分支自测无误后提交 PR 到 `master` 分支。
